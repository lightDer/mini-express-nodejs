# A minimum boilerplate for epxress API
- Node.js LTS 14.x.x

## covered
- Express.js
- Database integration (e.g. SQLite)
- Authentication
- Error handling and logging
- Config
- APIs documentation (Swagger)

## DEMO: a simple to-do list app
- POST /login
- DELETE /logout
- POST /todo
- GET /todos
- PUT /todo/:id
- DELETE /todo/:id
